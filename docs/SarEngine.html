引擎。

Syntax

var engine = new SarEngine();

Properties

fps
Number 表示当前引擎运行的帧速率。
status
EngineStatus 表示引擎的运行状态。
scene
Scene 引擎中活动的场景。

Methods

createByCanvas(canvas)
静态函数。在指定的Canvas上快速创建并初始化一个引擎实例，将自动创建包括输出接口、场景和摄像机等基础成分。
createInNode(element)
静态函数。在指定的HTML Element中快速创建并初始化一个引擎实例，将自动创建包括输出接口、场景和摄像机等基础成分。
end()
终止该引擎。
pause()
暂停该引擎的活动。
resume()
恢复该引擎的活动。
start()
启动该引擎。只有启动了该引擎，各种活动才会开始进行。

Events

onUpdate
在引擎更新游戏帧时触发。
onPause
在引擎暂停时触发。
onStart
在引擎开始运行时触发。
onResume
在引擎恢复运行时触发。
onEnd
在引擎终止时触发。
onError
在引擎运行过程中出现异常时触发。
